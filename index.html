🚀 Run Locally

Prerequisites

1. Install Ollama from https://ollama.ai/download
2. Pull the model: ollama pull deepseek-coder:6.7b

Backend (Express + WebSocket + AI)

cd backend
npm install
npx tsx src/index.ts
# → http://localhost:4000

Frontend (Next.js + Monaco Editor)

cd frontend
npm install
npx next dev --port 3001
# → http://localhost:3001/ghost-developer
